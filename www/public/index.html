Enabling Art

ARC is a render cluster that is free to use by any 3D artist. Our goal is to provide free, high power computing resources to artists anywhere. We plan to support Blender and LuxRender.

Our mission is to be completely free to use, however running the cluster will not be free. To support this project, please consider a donation. Your contributions will help get the project off the ground!


 

What is Arc?

ARC Render Cluster is an open-source and free to use render cluster designed to help 3D artists realize their creative dreams without having to maintain an expensive compute framework to render their creations.

ARC isn't quite up and running yet, mainly because of funding. In order to make our equipment purchases and support the project we need contrbutions from artists like you! If you would like to support this project, please consider a donation by clicking the button to the left.


Project Status

We'll keep you posted about the latest project happenings here.

Currently, we are still writing the web interface to the cluster. We have a development cluster set up and it is working with the files tested so far. What is left is we need to integrate it into the web interface and put in more tracking to allow users to cancel jobs and know how much cpu, memory, etc their job used.

Helping Others

Our mission is to help others in any way that we can. We are building ARC to help artists, but we will donate any remaning compute power to charities such as Folding at Home. Even if you aren't using the cluster, it still is helping others.
