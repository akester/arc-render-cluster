Enabling Art

ARC is a render cluster that is free to use by any 3D artist. Our goal is to provide free, high power computing resources to artists anywhere. We plan to support Blender and LuxRender.

Our mission is to be completely free to use, however running the cluster will not be free. To support this project, please consider a donation. Your contributions will help get the project off the ground!


Donate

ARC is provided free of charge, instead of charing to use this service, we rely on contributions from artists like you. If you have found this service useful or you would like to see the project succeed, please consider a donation.

The funding we get through donations as well as the revenue from ads on the site are our only sources of funding. Without donations, we would have no choice but to stop offering free service and offer the services for a fee, something that goes against the goals of this project.

Please consider giving what you can, we understand if you are unable to contribute. If you are able and willing, please consider supporting this project financially. You are enabling access to this cluster for any artist anywhere.

Project Status

We'll keep you posted about the latest project happenings here.

Currently, we are still writing the web interface to the cluster. We have a development cluster set up and it is working with the files tested so far. What is left is we need to integrate it into the web interface and put in more tracking to allow users to cancel jobs and know how much cpu, memory, etc their job used.

Helping Others

Our mission is to help others in any way that we can. We are building ARC to help artists, but we will donate any remaining compute power to charities such as Folding at Home. Even if you aren't using the cluster, it still is helping others.
